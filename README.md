# RI5_UML_Associacoes_POO
Projeto de POO em JavaScript baseado em um Diagrama UML, focado na implementação de classes com novas associações. Introduz a classe Funcionario e implementa a relação 1:N (Empresa para Funcionario). Utiliza a modularização ES6, incluindo uma classe DescritorEmpresa para gerar a saída formatada de todos os atributos da empresa e seus funcionários.
